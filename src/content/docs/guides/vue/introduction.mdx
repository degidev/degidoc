## Características clave de Vue 3

<Tabs>
  <TabItem label="Composition API">
    ```js
    import { ref, onMounted } from 'vue'

    export default {
      setup() {
        const count = ref(0)
        
        onMounted(() => {
          console.log('Componente montado')
        })

        return { count }
      }
    }
    ```
  </TabItem>
  <TabItem label="Options API">
    ```js
    export default {
      data() {
        return {
          count: 0
        }
      },
      mounted() {
        console.log('Componente montado')
      }
    }
    ```
  </TabItem>
</Tabs>

## Ventajas frente a Vue 2

<Aside type="tip" label="Mejoras">
- **Mejor rendimiento**: Sistema de reactividad reescrito
- **TypeScript**: Soporte nativo mejorado
- **Composition API**: Mejor reutilización de código
- **Fragmentos**: Múltiples elementos raíz
- **Teleport**: Renderizado en cualquier parte del DOM
</Aside>
